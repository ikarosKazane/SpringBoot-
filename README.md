# SpringBoot-需要源码可联系
本汽车出租综合管理系统采用前后端分离的架构模式，构建出一个灵活性以及可扩展性的系统架构。整个系统分为前端展示层、后端服务层和数据持久层三个主要部分，各层之间借助标准接口开展通信，实现了系统各部分的松耦合，有利于独立开展开发和维护工作。 前端展示层是基于 Vue.js 框架开发而成，采用组件化的开发方式，把用户界面拆解为可复用的组件，以此提升了代码的可维护性以及复用性，该层承担着页面渲染以及用户交互的职责，借助 Axios 库与后端服务进行通信，接收用户输入内容并展示处理结果，前端路由借助 Vue Router 得以实现，支持页面间无刷新切换。状态管理采用 Vuex，实现了组件间的数据共享，UI 组件库采用 Element UI，提供了统一的视觉风格以及交互体验。 
后端服务层基于 Spring Boot 框架构建，采用 RESTful API 设计理念，为前端提供标准化的数据接口，该层细分为控制器层、服务层以及数据访问层这三个子层。控制器层负责接收并处理 HTTP 请求，开展参数校验以及权限验证工作，服务层实现业务逻辑处理，封装核心业务规则，数据访问层凭借 MyBatis 框架与数据库进行交互，实现数据的增删改查操作，系统还集成了 Spring Security 组件，实现了用户认证以及授权管理。 数据持久层采用 MySQL 数据库，负责系统数据的持久化存储，数据库设计遵循规范化原则，依靠合理的表结构设计以及索引优化，提高了数据访问效率，系统运用连接池技术管理数据库连接，减少了连接创建以及销毁的开销，提升了系统性能，整个系统架构采用分层设计，各层职责清晰，边界明确，实现了高内聚、低耦合的设计目标。前后端借助 RESTful API 进行通信，使用 JSON 格式交换数据，便于不同平台的集成以及扩展，系统还预留了外部接口，可方便地与第三方系统进行对接，比如支付系统等。
